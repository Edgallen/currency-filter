### Установка проекта

Чтобы установкить все зависимости и запустить проект, нужно выполнить следующие действия:

1. Клонировать репозиторий
   ```sh
   git clone https://github.com/Edgallen/currency-filter
   ```
2. Установить зависимости через npm
   ```sh
   npm install
   ```
3. Запустить проект
   ```sh
   npm start
   ```
